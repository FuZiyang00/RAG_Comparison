# RAG_Comparison

Comparing a "traditional" RAG with Microsoft's framework Graph RAG. 

## Benchmark
3 GMAT Reading comprehensions passages (source: https://gmatclub.com/forum/100-hardest-reading-comprehension-passages-questions-424608.html) and 10 multiple choices questions on their content.
